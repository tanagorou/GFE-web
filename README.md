# GFE-web

> **ポモドーロ学習を“PCだけ”で、途切れず回せる Web アプリ**

スマホに触れずに作業へ没頭できるよう、「作業 → 休憩 → 作業…」を自動ループ。通知・サウンド設定も稼働中に変更できます。  
直感的でシンプル、学習ログはグラフで振り返り。

---

## 使用技術

- **Backend**: Rails 7.2  
- **Frontend**: TypeScript  
- **DB**: PostgreSQL  
- **Infra**: Docker / Render（デプロイ）  
- **その他**: jsbundling / esbuild / importmap など（構成に応じて）

---

## 背景

スマホのポモドーロは便利だけど、**つい YouTube やゲームを開いてしまう問題**がありました。  
「**PC だけで完結**」「**作業／休憩が自動で切り替わる**」仕組みが欲しくて、このアプリを作りました。

---

## 特長（できること）

- **一度スタートで自動ループ**  
  作業・休憩を設定したら、**停止するまで永遠にポモドーロが回り続けます**。
- **直感操作**  
  学習管理アプリにありがちな多機能・複雑さを排除。**必要最小限で迷わない UI**。
- **学習の軌跡をグラフ化**  
  実行履歴を可視化して、モチベーションと振り返りに。
- **通知・サウンドをリアルタイム変更**  
  **稼働中でも設定変更**が反映されます。
- **バックグラウンドでのタイマー、通知、音の動作**  
  ebページ上でのページ遷移があってもタイマーは動作し続け、またバックグラウンドでもタイマーは動作、通知、音が鳴ります
